### ⌛ 开始

#### 创建虚拟环境

* 在根目录下创建虚拟环境

#### 安装第三方库

```
pip install flask==3.1.0 -i https://pypi.tuna.tsinghua.edu.cn/simple
```

#### pip 国内镜像源
